# Scripts — TechnoJaw App

| Script | Comando | Descrizione |
|--------|---------|-------------|
| `generate-icons.py` | `npm run icons` | Genera tutte le icone da `assets/icon-master.png` |

- Python 3.8+
- Pillow (`pip3 install Pillow`)
