# Assets — TechnoJaw App

## Contenuto

| File | Dimensione | Descrizione |
|------|-----------|-------------|
| `icon-master.png` | 1024×1024 | Sorgente unica per tutte le icone |

## Regole

1. **`icon-master.png`** è l'UNICA sorgente — tutto il resto è generato
2. Per rigenerare: `npm run icons` (usa `scripts/generate-icons.py`)
3. Le icone finali vanno in `src-tauri/icons/` — **NON editare manualmente**
4. Formato richiesto: PNG, RGBA, esattamente 1024×1024 px
